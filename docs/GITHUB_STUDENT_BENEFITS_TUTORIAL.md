# GitHub 学生开发者福利包申请教程

## 📚 什么是 GitHub 学生开发者福利包？

GitHub Education 为在读学生提供免费的开发者工具和服务包，价值超过 **$200,000**，包括：

- **GitHub Pro** 账户（免费）
- **免费域名** (.me 域名一年）
- **云服务额度**（AWS, Azure, DigitalOcean 等）
- **开发工具**（Copilot, JetBrains 全家桶等）
- **设计工具**（Canva Pro, Figma 等）
- **更多学习资源**

---

## ✅ 申请资格要求

1. **年满 13 岁**
2. **目前在读学生**（中学、大学、职业学校等）
3. **拥有 GitHub 账号**
4. **可验证的学校电子邮件地址** 或 **学生证/在读证明**

---

## 📝 申请步骤

### 步骤 1：准备必需材料

在开始申请前，请准备以下材料：

✅ **学校电子邮件地址**（推荐）
   - 例如：`your-name@student.school.edu.my`
   - 必须是学校提供的官方邮箱

**或者**

✅ **学生证或在读证明**
   - 需要清晰显示：
     - 你的姓名
     - 学校名称
     - 有效日期/当前学期
     - 学校官方印章或标识

### 步骤 2：创建或登录 GitHub 账号

1. 访问 [https://github.com](https://github.com)
2. 如果已有账号，直接登录
3. 如果没有账号：
   - 点击右上角 **Sign up**
   - 输入邮箱、密码、用户名
   - 完成验证

### 步骤 3：添加学校邮箱到 GitHub 账号

1. 登录 GitHub 后，点击右上角头像
2. 选择 **Settings**（设置）
3. 左侧菜单选择 **Emails**
4. 点击 **Add email address**
5. 输入你的学校邮箱地址
6. 点击 **Add**
7. 前往你的学校邮箱，查收验证邮件
8. 点击邮件中的验证链接

### 步骤 4：访问 GitHub Education 申请页面

1. 访问：[https://education.github.com/benefits](https://education.github.com/benefits)
2. 点击绿色按钮 **Get student benefits**（获取学生福利）

### 步骤 5：填写申请表单

#### 5.1-选择教育身份

- 选择 **Student**（学生）

#### 5.2 填写基本信息

- **Name**（姓名）：输入你的真实姓名
- **Email**（邮箱）：
  - 从下拉菜单选择你的学校邮箱
  - 如果没有学校邮箱，可以使用个人邮箱，但需要上传证明文件

#### 5.3 填写学校信息

- **School name**（学校名称）：
  - 输入你的学校名称（英文）
  - 例如：`Kuching Chinese High School`
  - 会自动搜索匹配的学校，如果找到请选择
  
- **Graduation year**（毕业年份）：
  - 选择你预计的毕业年份

#### 5.4 说明用途

- **How do you plan to use GitHub?**（你计划如何使用 GitHub？）
  - 用 **英文** 简短描述（至少 50 字符）
  - 示例回答：
    ```
    I am a computer science student learning web development and programming. 
    I plan to use GitHub to:
    - Host my coding projects and assignments
    - Collaborate with classmates on school projects
    - Learn from open-source projects
    - Build my portfolio for future career opportunities
    - Practice version control and team collaboration skills
    ```

#### 5.5 上传证明文件（如果需要）

如果你使用的不是学校邮箱，需要上传证明文件：

1. 点击 **Upload proof of academic status**
2. 选择文件类型：
   - **Academic ID**（学生证）
   - **Dated school transcript**（带日期的成绩单）
   - **Dated admission letter**（带日期的录取通知书）
   - **Other**（其他）
3. 拍摄或扫描文件，确保：
   - **清晰可见**
   - **包含你的姓名**
   - **包含学校名称**
   - **显示当前有效日期**
   - **包含学校印章或标识**
4. 上传文件（支持 JPG、PNG、PDF，最大 2MB）

### 步骤 6：提交申请

1. 勾选 **"I agree to the Terms of Service"**（同意服务条款）
2. 点击 **Submit your information**（提交信息）

---

## ⏳ 等待审核

- **处理时间**：通常 **15 分钟 - 7 天**
- **自动批准**：如果使用学校邮箱，通常会自动批准（几分钟内）
- **人工审核**：需要上传证明文件的申请需要人工审核（1-7 天）

### 审核状态查询

1. 访问：[https://education.github.com/discount_requests](https://education.github.com/discount_requests)
2. 查看申请状态

---

## ✅ 批准后的操作

### 1. 确认福利激活

收到批准通知后：
1. 访问：[https://education.github.com/pack](https://education.github.com/pack)
2. 你应该看到绿色勾号 ✓ Student Developer Pack
3. 点击 **Get your pack**

### 2. 激活各项福利

滚动页面查看所有可用的福利，重要的包括：

#### 🔥 必备福利：

1. **GitHub Copilot**（AI 编程助手）
   - 点击 **Get access to GitHub Copilot**
   - 自动为你的账号激活

2. **GitHub Pro**（专业版）
   - 自动激活
   - 获得无限私有仓库
   - 高级统计和洞察

3. **DigitalOcean**（云服务器）
   - $200 信用额度
   - 点击 **Get access to DigitalOcean**
   - 注册账号并验证

4. **Namecheap**（域名）
   - 免费 .me 域名一年
   - 免费 SSL 证书
   - 点击 **Get access to Namecheap**

5. **Canva Pro**（设计工具）
   - 免费 12 个月
   - 点击 **Get access to Canva**

6. **JetBrains**（开发工具全家桶）
   - 包括 IntelliJ IDEA, PyCharm, WebStorm 等
   - 点击 **Get access to JetBrains**

## ❓ 常见问题

### Q1: 我没有学校邮箱怎么办？
**A**: 上传学生证或在读证明。确保文件清晰，包含必要信息。

### Q2: 申请被拒绝了怎么办？
**A**: 
- 检查提交的信息是否准确
- 确保证明文件清晰可见
- 重新提交申请，并在说明中提供更多详细信息

### Q3: 学生福利有效期多久？
**A**: 
- 只要你还是学生，可以每年续期
- 毕业后失效
- 需要每 1-2 年重新验证学生身份

### Q4: 可以使用私人邮箱申请吗？
**A**: 可以，但需要上传额外的证明文件，审核时间会更长。

### Q5: 申请费用多少？
**A**: **完全免费**！GitHub Education 是免费为学生提供的。
